URL Shortener Using Django

The Django URL Shortener as the name suggest is a URL shortening service which was completely built from scratch by me. It uses a base 62 to decimal conversion to convert a long URL to a short one.

The URL shortener works on the Base62 algorithm and it takes a number which is the primary key of the long URL in the databse and then converts it to base62 and gives us the short URL. The short URL is again converted to long URL using the base62 to Decimal algorithm.
